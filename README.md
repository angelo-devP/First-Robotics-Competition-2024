Código que foi utilizado no ano de 2024 pela equipe Forest Guardians #10295.

⚠️ **Aviso Importante:**  
Este código pode não funcionar corretamente atualmente, pois é um projeto mais antigo e as bibliotecas da REV, WPILib e de outros componentes já foram atualizadas ao longo do tempo.

Este repositório tem como objetivo servir como referência e histórico de desenvolvimento da equipe.
